##  &#x1F375;文件说明

- `doc` 中包括 ANSYS, ABAQUS, LS-DYNA 的资源书籍，大于 100MB 的采用了分卷压缩，下载完成后，直接点击 `.zip` 文件解压缩即可生成完成的文件内容；
- `code` 为对应书籍中的部分代码；
- `mac` 为数据后处理的宏文件内容；
  - `array2csv_disp.mac`：导出节点位移数据到 `.csv` 文件；
  - `extra_strain.mac`：提取整体结构每个收敛子步的最大应变；
  - `get_picture.mac`：截图设置；
